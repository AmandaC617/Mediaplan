如何取得 API 金鑰：

  1. 前往 Google AI Studio
  2. 登入您的 Google 帳戶
  3. 點擊「Create API Key」建立新的 API 金鑰
  4. 複製 API 金鑰並貼上到上方欄位

注意：API 金鑰格式應為類似「AIzaSyC...」的長字串。如果遇到「API key not valid」錯誤，請確認：

  • API 金鑰是否完整複製（沒有遺漏字符）
  • 是否已啟用 Gemini API 服務
  • API 金鑰是否有足夠的配額
  • 是否在正確的地區使用（某些地區可能有限制）
